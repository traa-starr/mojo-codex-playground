# Mojo Codex Playground

A minimal React + Tailwind digital art page with floating philosophical text and ambient motion.

## Run locally

```bash
npm install
npm run dev
```

Then open the Vite URL (default: `http://localhost:5173`).

## If files are not showing in GitHub

Make sure you have pushed this branch to your GitHub remote:

```bash
git remote add origin <your-repo-url> # if not already configured
git push -u origin work
```
